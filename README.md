# Book_My_Show
Built a Book My show Application where admin can add Show, Theater, Movie, User can Book Ticket  and can cancel Ticket..
#Technology Used 
Java
Spring Boot
Hibernate
RESTful APIS
Maven
My SQL
Postman 
#Functionality
User
• Add User
Theater
• Add Theater
Movies
• Add Movies
Shows
• Add Shows
Tickets
• Book Tickets
  ○ Send email notification to the user
  
• Cancel Tickets
  ○ Send email notification to the user
